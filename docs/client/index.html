Prerequisites

Technology: Version

Docker: 27.2.0

Node: 22.9.0

NPM: 10.9.1

Commands

If you want to start the project on the local machine use this command:

npm start

It will start the project in development mode and therefore will use the development constants.

Environments setup

You need to add a .env.local file. For this you could just copy the .env.dev.local file and rename it.

Docker setup

If you want to run the project in docker use this command:

    docker compose up -d

This will create the project under http://localhost:8080/

But keep in mind, that it will then send all API calls to the production backend.

Production

The project is reachable under the url: https://advanced-client.goldenpc.ch/
